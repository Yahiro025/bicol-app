# Product

## Register

product

## Users
Every Bicolano, including the diaspora, students in the Bicol region, and language learners. The context is someone seeking quick, authoritative translations or engaging in long-term vocabulary building through learning modules.

## Product Purpose
A professional, full-stack dictionary platform that bridges academic precision with modern accessibility. It exists to preserve and promote the Bikol language through AI-enriched data, multi-dialect support, and cross-platform utility.

## Brand Personality
**Hybrid: Academic Authority + Community Warmth**
- **Trustworthy**: AI data and definitions are presented with precision and cleanliness.
- **Approachable**: UI elements feel friendly, encouraging, and human.
- **Dignified**: Modern presentation that respects the cultural weight of the language without feeling sterile or childish.

## Anti-references
- **Sterile Machine Translation**: Avoid the cold, rigid feel of a generic Google Translate clone.
- **Web 1.0 Text Wall**: Avoid unbroken blocks of text; use modern hierarchy and spacing instead of traditional dictionary layouts.
- **Overly Gamified/Childish**: Avoid elementary worksheet aesthetics or aggressive "Duolingo-style" gamification that compromises dignity.

## Design Principles
- **Data Precision, Chrome Comfort**: Present core language data with academic clarity (high contrast, clean grids) while wrapping it in warm, approachable UI chrome (rounded corners, bouncy interactions).
- **Modern Cultural Dignity**: Use sophisticated visual hierarchy and subtle borders to create a tool that feels like a modern cultural artifact, not just a database.
- **Encouraging Authority**: Use tone and copy that invites contribution ("Contribute") and learning ("Learn Bikol") while maintaining trust in the accuracy of the enrichment.
- **Task-Focused Density**: Allow for information-rich word entries while using layout to prevent cognitive overload.

## Accessibility & Inclusion
- **WCAG AA Compliance**: Ensure high-contrast legibility, especially for zinc-400/500 text against zinc-950 backgrounds.
- **Semantic Dictionary Structure**: Use `<dl>`, `<dt>`, and `<dd>` for entries and ARIA labels for functional pills (e.g., "Synonym: [word]").
- **Motion Sensitivity**: Respect `prefers-reduced-motion` by simplifying Framer Motion animations to fades.
- **Physical Accessibility**: Minimum 44x44px touch targets for all interactive elements (A-Z filters, category pills, links).
